- `index.html`
- `styles.css`

## Deploy on Vercel

1. Create a new GitHub repository.
2. Upload `index.html` and `styles.css`.
3. Go to Vercel.
4. Click **New Project**.
5. Import the GitHub repository.
6. Keep framework preset as **Other** or **No Framework**.
7. Deploy.

Your live page will look like:

`https://your-project-name.vercel.app`

You can then add that URL to your beehiiv website navigation or link to it from a beehiiv custom page/button.
